## How to Play

1. Click "Start Game" to begin
2. Use **Arrow Keys** to control the snake direction
3. Eat the red food to grow and gain points
4. Avoid hitting the walls and yourself
5. Try to beat your high score!

## Controls

- **Arrow Up**: Move up
- **Arrow Down**: Move down
- **Arrow Left**: Move left
- **Arrow Right**: Move right
- **Start Button**: Start or restart the game
- **Pause Button**: Pause/Resume the game

## Installation

Simply open `index.html` in your web browser to play.
